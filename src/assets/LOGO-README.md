# VARZEANDO - Logo Pack

Este diretório contém todos os assets de marca do VARZEANDO.

## 📦 Arquivos Disponíveis

### Logos Principais
- **varzeando-logo.png** - Logo completo horizontal (sem bola de futebol)
- **varzeando-logo-with-ball.png** - Logo completo horizontal com bola de futebol
- **varzeando-icon.png** - Ícone/símbolo (apenas letra "V")

## 🎨 Especificações

### Estilo Visual
- **Estilo**: Grafite/Pixo (street art brasileiro)
- **Cor Principal**: Laranja Várzea (#E87400)
- **Contraste**: Preto/Branco para contornos
- **Atmosfera**: Autêntica, popular, energética

### Uso Recomendado

#### Logo Completo (varzeando-logo.png)
- Header do website
- Documentação oficial
- Apresentações formais
- Quando há espaço horizontal suficiente

#### Logo com Bola (varzeando-logo-with-ball.png)
- Marketing e redes sociais
- Material promocional
- Quando quiser enfatizar o contexto de futebol
- Campanhas publicitárias

#### Ícone (varzeando-icon.png)
- Favicon do site
- App mobile (ícone do app)
- Avatares em redes sociais
- Quando o espaço é limitado (quadrado)

## 📐 Dimensões

- **Logo Horizontal**: 1920x640px (3:1 ratio)
- **Ícone**: 1024x1024px (1:1 ratio)

## 🎯 Paleta de Cores

```css
--primary: #E87400;        /* Laranja Várzea */

--success: #38761D;        /* Verde Gramado (autenticidade) */
--white: #FFFFFF;

```

## 📝 Diretrizes de Uso

### ✅ Permitido
- Usar em materiais oficiais do VARZEANDO
- Redimensionar mantendo proporções
- Usar em fundos claros ou escuros (design adaptável)

### ❌ Não Permitido
- Alterar as cores da marca
- Distorcer ou esticar o logo
- Adicionar efeitos não autorizados
- Usar versões de baixa qualidade
